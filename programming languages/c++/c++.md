# String and Vector and Array Size

```
// String #include <string>
string s = "test";
s.length(); // 4 

// Vector #include <vector>
vector<int> nums;
nums.size(); //0

//Array
int nums[4] = {1, 2, 3, 4};
nums.length(); // 4 
```

# Sorting an Array

```
// #include <algorithm>
void SortList(std::vector<int> nums)
{
    std::sort(nums.begin(), nums.end());
}
```
